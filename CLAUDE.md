# 全局硬规则锁

## 世界观物理规则
1. **灵气潮汐定律**：灵气以12年为一个周期涌动，每逢涌动则修为易突破
2. **能量守恒法则**：所有超凡力量消耗等价生命力或精神力
3. **因果追溯律**：改变重大历史事件必受天道反噬
4. **领地锚定律**：修炼者建立洞府后与此地气运绑定

## 主角永久性格定义
- 核心：利己但不损人，精明但不奸诈
- 行为准则：人不犯我我不犯人，人若犯我加倍奉还
- 情感线：多情不滥情，责任肩上扛
- 禁忌：挚友被伤必报复，家人被辱不惜命

## 文风禁止列表
- 禁止使用"读者老爷"、"作者菌"等打破第四墙的表达
- 禁止超过300字的纯内心独白
- 禁止战斗场景中使用超过5个技能名连续堆砌
- 禁止在单章内让主角陷入绝境超过3次（过度虐主）

## 伏笔永久保护区 ID 前缀规则
- FORESHADOW_MAIN_*：主线核心伏笔，禁止在第三卷前强制收束
- FORESHADOW_FEMALE_*：女主相关伏笔，禁止在其他女主场景收束

## 叙事底线
- 主角不得主动背叛国家和民族利益
- 主角不得对未成年人下手
- 主角不得参与贩毒、人口贩卖
- 主角可以对敌人使用任何手段

## 质量标准
- 每章至少2个有效信息点（推进剧情或展现人物）
- 对话不超过单章40%
- 战斗场景必须有明确的能力展示和结果
- 每10章必须有至少1个小高潮或反转
