## 4. 作品简介

```
叶尘重生回灵气复苏前夕，觉醒混沌修炼系统，修上古混沌诀。

上一世他在末世中挣扎十年，见证生灵涂炭、山河破碎，最终依然无力回天。重生归来，他誓要改写命运，阻止悲剧重演。

叶家庶子，叶承宗迫害，险象环生。偶获上古大能遗留的混沌玉简，其中蕴含本源之力。他从街头拾荒起步，第一桶金、古玩捡漏、拜师老周，一步步崛起于微末。

随着灵气潮汐爆发，修炼界格局大变。黑暗势力蠢蠢欲动，七大宗门明争暗斗，四大家族尔虞我诈。叶尘护佳人、斩仇敌、破阴谋，在末世降临前布局天下。

混沌初开，乾坤未定。飞升境的屏障正在松动，属于他的传奇才刚刚开始。
```
